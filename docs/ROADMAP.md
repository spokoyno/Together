# Roadmap

## Phase 0 — foundation

Запуск starter, Git, Supabase, Vercel и документация.

## Phase 1 — identity

Auth, профиль, пара, приглашение, отвязка.

## Phase 2 — daily loop

Dashboard, настроение, вопрос дня.

## Phase 3 — utility

Планы, события, обратный отсчёт.

## Phase 4 — memories

Лента, приватные фото, теги.

## Phase 5 — closed test

3–5 пар, интервью, исправления, аналитика минимального уровня.

## Decision gate

Следующая версия создаётся только после ответа:
- возвращаются ли оба партнёра;
- какие две функции используются чаще всего;
- что люди считают неудобным;
- какая функция действительно удерживает пару.
