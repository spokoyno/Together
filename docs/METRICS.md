# Pre-MVP metrics

## Главные события

- signup_completed;
- pair_created;
- invitation_accepted;
- mood_shared;
- plan_created;
- plan_completed;
- question_answered;
- memory_created;
- returned_next_day;
- returned_next_week.

## Главный показатель

Вернулись ли оба партнёра, а не только создатель пары.

## Безопасная аналитика

Не отправлять в аналитику:
- текст ответов;
- заметки настроения;
- названия личных планов;
- содержимое воспоминаний;
- медиа;
- email.

На первом закрытом тесте можно обойтись интервью и агрегированными счётчиками.
